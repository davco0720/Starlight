# Starlight

## Ejecución del Juego
Para ejecutar el juego, abre el proyecto en NetBeans y ejecuta la clase Main.

## Controles del Juego
- **Flechas del teclado:** Mover al personaje.
- **A,W,D:** Mover al personaje.
- **E:** Pelear (Solo último nivel).
- **Espacio:** Saltar.
- **Escape:** Pausar el juego.
